Fetch Quotes

All you need to do is send a request to https://inspiroquote.me/api/[mode], where:

  • [mode] = retreval type [catgory, author, daily_quote]

Sample Requests

  • All Quotes: https://inspiroquote.me/api/v1/quotes - Get all quotes
  • Quote of the Day: https://inspiroquote.me/api/v1/daily_quote - Get the quote of the day
  • All Categories: https://inspiroquote.me/api/v1/categories - Get all categories
  • Quotes by Category: https://inspiroquote.me/api/v1/categories/[id]/quotes - Get quotes by category
  • All Authors: https://inspiroquote.me/api/v1/authors - Get all authors
  • Quotes by Author: https://inspiroquote.me/api/v1/authors/[id]/quotes - Get quotes by author
